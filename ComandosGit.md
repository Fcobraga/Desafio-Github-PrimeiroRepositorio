#Comandos

- cd : Usado para mudar de pasta.
- cd .. : Usado para volta a pasta anterio
- mkdir : Usado para cria pasta/arquivos
- del/rmdir : Usado para deleta pasta/arquivo
- tab : Completa sentença
- control+l : Limpa tela
- setas : navega nos comandos
- Git int : Cria um repozitorio
- ls : Mostra o local atual
- echo :  Cria um novo arquivo
- Git status : Mostra alterações que fora feitas e n estão salvas no terminal
